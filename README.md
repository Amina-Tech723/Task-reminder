A user-friendly "Task Reminder" application developed using "HTML, CSS, and JavaScript". This project helps users create, organize, and manage daily tasks efficiently. It provides an interactive interface to add, update, and delete tasks while keeping track of important reminders.

Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Responsive and clean user interface
- Easy-to-use task management
- Interactive user experience using JavaScript

 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

Project Structure


Task-Reminder/
│── index.html
│── style.css
│── script.js
│── images/
└── README.md


 Preview

Open the `index.html` file in your browser to use the application.

 How to Run

1. Clone the repository:
   bash
   git clone https://github.com/Amina-Tech723/Task-Reminder.git
   

2. Open the project folder.

3. Run the project by opening `index.html` in any web browser.
